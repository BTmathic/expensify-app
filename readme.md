# Expensify

Expense tracker with authentication through Google to keep all user expenses private. Allows users to enter/edit/remove expenses as well as filter by string (e.g., 'rent') or dates.

### Front End

  * React

### Back End

  * Redux
  * Firebase

### Build Tools

  * Parcel
  * Babel
  * Yarn

### Testing

  * Enzyme
  * Jest
